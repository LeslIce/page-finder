# Page Finder 📖

一个轻量的单词笔记本索引工具，用于记录和查找手写单词本中的词条。

## 功能

- **多笔记本管理**：支持创建多个笔记本，对应实体单词本的不同册
- **词条录入**：记录单词及其所在页码，支持标记"辨析词"类别
- **全局搜索**：跨笔记本搜索，点击结果跳转定位并高亮
- **排序与筛选**：按页码或时间排序，可筛选仅显示主词条或全部
- **左滑操作**：iOS 风格的左滑手势，露出编辑和删除按钮
- **数据备份**：支持全量导出为 JSON 文件，以及从备份文件导入恢复
- **AI 释义**：搜索结果可调用 API 自动获取单词释义（需自行配置）

## 技术

纯前端单文件实现，无框架依赖。数据存储于浏览器 IndexedDB，部署于 GitHub Pages。

## 使用

访问 [leslice.github.io/page-finder](https://leslice.github.io/page-finder/) 即可使用。建议添加到 iPhone 主屏幕以获得最佳体验。

## 备份提醒

数据保存在浏览器本地，清除浏览器数据会导致记录丢失。请定期通过设置页面的「导出数据」功能进行备份。
